The new LLC replacement policies can be implemented by using the updated base_replacement.cc file (which is to be put in the replacement folder along with the new policy files).

The functions should be defned in the block.h and cache.h files in the inc folder( the new updated bloch.h and cache.h can be replaced in the inc folder).

The result simulations can be generated by pluggin in the required parameters in the commands.sh file

The trace files used are gcc_39B.trace.xz, , astar_163B.trace.xz and tonto_2834B.trace.xz.
